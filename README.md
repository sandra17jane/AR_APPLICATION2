# AR Computer Hardware Learning App

This project is an Augmented Reality (AR) based Android application that helps users learn computer hardware components through interactive 3D models.

## Features
- AR-based visualization of computer hardware
- Interactive 3D component exploration
- Educational learning support
- User-friendly Android interface

## Technologies Used
- Android Studio
- Java / Kotlin
- ARCore
- Blender (for 3D models)
